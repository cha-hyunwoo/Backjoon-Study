## 📂 폴더 구조 설명

이 저장소의 코드는 작성 시기에 따라 두 가지 폴더로 나뉩니다.

### 1. `백준/` (Active)
- **현재 사용 중**인 폴더입니다.
- 백준 봇(Chrome Extension 등)을 통해 **자동으로 업로드**된 최신 문제 풀이들이 저장됩니다.
- 문제 분류별/난이도별로 정리되어 있습니다.

### 2. `src/` (Legacy)
- **과거에** 제가 직접 문제를 풀고 **수동으로 업로드**했던 코드들입니다.
- 현재는 업데이트되지 않으며, 기록 보관용으로 남겨두었습니다.
